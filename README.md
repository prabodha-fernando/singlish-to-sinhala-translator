# Singlish to Sinhala Transliteration App & Testing

This project is a web-based application that converts Singlish input into Sinhala text in real-time. It includes a premium UI and a specialized word-filtering system to explore available test cases.

## Project Structure
- `index.html`: The main web application.
- `Positive.spec.js`: Playwright automated tests for 24+ positive scenarios.
- `Negative.spec.js`: Playwright automated tests for 10+ negative scenarios.
- `package.json`: Project metadata and dependencies.

## Prerequisites
Before running the tests, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (Latest LTS version)
- [Git](https://git-scm.com/)
